We're proud to announce today the release of UnityVS 1.0.

UnityVS is a commercial product of SyntaxTree. UnityVS is a bridge between Unity, a fantastic tool for video game development or interactive content creation, and Visual Studio, the renowned development environment from Microsoft.

UnityVS is packed with useful features for the Unity programmer. With UnityVS you can debug your games from Visual Studio while they are running in the Unity Editor: just press F5 to attach the Visual Studio Debugger to Unity without interrupting your workflow! UnityVS also adds support to Visual Studio to all the programming languages that you can use to program Unity: UnityScript and Boo.

As of today, UnityVS requires Microsoft Visual Studio 2010 and Unity 3.5 on Windows. We're already at work to add support for new versions, features and platforms!
