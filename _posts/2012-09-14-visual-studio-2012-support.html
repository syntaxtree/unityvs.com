--- layout: post title: Visual Studio 2012 Support date: 2012-09-14 15:30:00 ---

Two days ago, Visual Studio 2012 was officially launched.

Today, we release the support of Visual Studio 2012 with UnityVS 1.0.7.

You can have both UnityVS 2010 and UnityVS 2012 installed. Just make sure you import the proper version of our Unity Package inside your Unity project, and you'll be good to go.

Unity Pro customers can now have a Visual Studio that matches beautifully with Unity.
