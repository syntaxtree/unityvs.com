--- layout: post title: New productivity features ---

Not only we have been fixing bugs in the beta, but we're also implementing new productivity features based on beta testers feedbacks.

Unity Project Explorer

Take for instance the “Unity Project Explorer” view, it's quite handy as it mirrors the view of your project that you have inside Unity. But now you can even dive into the different scripts and directly go to the script function you're interested in.

Script method wizard

When writing a Unity script, be it in C#, UnityScript or Boo, you need to implement special methods that are recognized by Unity. You can now invoke the method creation wizard with a few keystrokes, and select the methods to add to your script.

Quick script method creator

If you know exactly the script method you want to implement, press ctrl+shift+q to open the quick script method window. Quickly type the method name by using camel bumps (inspired by Resharper's Goto-Type or Goto-Member functionality) to create a script method stub.
