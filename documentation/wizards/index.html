Scripting Wizards

MonoBehavior Wizard

Especially helpful when learning the Unity API, the MonoBehavior wizard let you visually pick which Unity functions to implement in your MonoBehavior.

Shortcut: CTRL+SHIFT+M

Quick MonoBehavior

For those who already know which functions they need to implement, the quick MonoBehavior wizard lets you implement a MonoBehavior function quickly, especially when you use camel humps. For instance, you can input OMD, and the wizard will match OnMouseDown and OnMouseDrag.

Shortcut: CTRL+SHIFT+Q
