Log Callback

UnityVS registers its own log callback to be able to stream the Unity console to Visual Studio. If you have editor scripts that were previously registering their own callback, we can all work together if you plug into our own log callback.

{% highlight csharp %} using System; using UnityEngine; using UnityEditor; using SyntaxTree.VisualStudio.Unity.Bridge; [InitializeOnLoad] public class LogCallbackHook { static LogCallbackHook() { VisualStudioIntegration.LogCallback += (string condition, string trace, LogType type) => { // ... }; } } {% endhighlight %}
