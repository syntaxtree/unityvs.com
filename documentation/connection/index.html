Connection between Visual Studio and Unity

UnityVS shows whether it is connected to a Unity instance or not

Unity and UnityVS are talking through a udp connection, if your firewall asks about them, authorize them.

If UnityVS failed to connect to a Unity instance, its status icon will be greyed out. Click on it to select a Unity instance to connect to.
