F.A.Q. よくある質問と回答

技術面

1. UnityVS が対応している Visual Studio のバージョンは何でしょうか？

UnityVS は、Visual Studio 2010 および Visual Studio 2012 に対応しています。フル機能版の Visual Studio が必須であり、具体的には Visual Studio Professional 以上のエディションが必要です。

2. Visual Studio Express では UnityVS を利用できないのでしょうか？

残念ながら Visual Studio Express はプラグインに非対応ですので、UnityVS をご利用頂くことはできません。

3. UnityVS を利用したいのですが、Visual Studio を購入するための予算がありません。どうすれば良いでしょうか？

マイクロソフト社では、マイクロソフト社製の開発ツール群を利用するための様々な支援プログラムを提供しています。学生の方であれば、在籍中の学校が MSDN AA (MSDN アカデミック アライアンス) プログラムに参加しているかも知れません。その場合はそれを活用すると良いでしょう。またマイクロソフト社は、小規模企業および新興企業を対象とした BizSpark や WebsiteSpark といったプログラムを提供しています。

4. UnityVS が対応している Unity のバージョンは何でしょうか？

UnityVS は、Windows 版の Unity 3.5 および Unity 4 に対応しています。無料版の Unity でも Unity Pro でもご利用頂けます。

5. UnityVS が対応しているプログラミング言語は何でしょうか？

UnityVS は、C# 3、UnityScript および Boo に対応しています。

6. UnityVS は、各言語にローカライズされていますか？

現段階では、UnityVS は英語版のみとなっています。
